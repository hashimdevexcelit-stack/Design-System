System Overview

Figma-style documentation, not a marketing webpage.

This version is organized like a clean internal design system board: section heading, system content, then the next section. The original RedBird color language, buttons, fields, metrics, and modal tone remain the same. Missing supporting pieces such as toasts, badges, tabs, empty states, and richer data patterns are added in a modern but compatible style.
